# 🐱🐶 Cat vs Dog Classification using MobileNetV2

This project is an AI-powered image classification web application that predicts whether an uploaded image is a **cat** or a **dog** using Deep Learning and Transfer Learning techniques.

The model is built using TensorFlow and MobileNetV2, and the application is deployed using Streamlit and Render.

---

## 🚀 Live Demo

https://cat-vs-dog-classifier-qbwe.onrender.com

---

## 📌 Features

- Upload image for prediction
- Predict whether the image is a cat or dog
- Display confidence score
- Real-time image classification
- Deep Learning powered web application
- Publicly deployed AI application

---

## 🧠 Technologies Used

- Python
- TensorFlow
- MobileNetV2
- Transfer Learning
- Streamlit
- Render

---

## 📊 Model Performance

- Achieved approximately **98–99% validation accuracy**
- Used Transfer Learning with MobileNetV2 for better performance and faster training
- Optimized for lightweight and efficient image classification

---

## 🖼️ Project Workflow

1. Image Upload
2. Image Preprocessing
3. Feature Extraction using MobileNetV2
4. Model Prediction
5. Display Final Result with Confidence Score
