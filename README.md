# Cat vs Dog Classification using MobileNetV2

This project is an AI-powered image classification system that predicts whether an uploaded image is a cat or a dog.

## Technologies Used

- TensorFlow
- MobileNetV2
- Transfer Learning
- Streamlit
- Python

## Features

- Upload image
- Predict cat or dog
- Confidence score
- Deep Learning model deployment

## Model Accuracy

Achieved approximately 98-99% validation accuracy.
